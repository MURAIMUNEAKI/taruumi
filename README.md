# 地域のチカラ ＆ 烏山ご近所めぐり デモアプリ

本プロジェクトは、高校生と大学生の自由な発想や行動力を生かした地域共創メディア「地域のチカラ」のWebアプリケーション・デモサイトです。

一般公開に向けたデザイン検証用、およびフロントエンド開発の学習用に構築されたシンプルなHTML/JS/CSSのコードベースとなっています。

---

## 📁 ファイル構造

リポジトリ内の主要なファイルとその役割は以下の通りです。

```bash
taruumi/
├── README.md           # 本ドキュメント（プロジェクトの概要・解説）
├── index.htm           # トップページ（地域のチカラトップ ＆ 烏山ご近所めぐり詳細）
├── kiji.htm            # 記事一覧（生産者インタビューやコラム）
├── chokubai.htm        # 直売所一覧（農産物直売所の営業時間や情報）
├── denwacho.htm        # 農家・お店 電話帳（連絡先カード一覧）
├── event.htm           # イベント・募集一覧（マルシェや清掃ボランティア等）
└── report.htm          # 活動レポート（共同プロジェクトの活動報告記録）
```

## 💡 学生向け・フロントエンド学習のポイント

このデモサイトには、モダンなフロントエンド開発の基礎となるテクニックが多数散りばめられています。コードを読む際、以下のポイントに注目して学習してみましょう。

### 1. 2つの画面遷移スタイル（SPA風 ＆ MPA風）
- **SPA（シングルページアプリケーション）風の切り替え**:
  `index.htm` では、トップの「地域のチカラ」と詳細の「烏山ご近所めぐり」を、別のファイルに分けるのではなく、JavaScript（`showView` 関数）でHTML要素の `hidden` クラスを付け外しすることで、瞬時に画面が切り替わる仕組み（擬似的な画面遷移）を作っています。
- **MPA（マルチページアプリケーション）風の切り替え**:
  「記事」「直売所」「電話帳」「イベント・募集」「活動レポート」はそれぞれ別ファイル（`kiji.htm`, `chokubai.htm`, `denwacho.htm`, `event.htm`, `report.htm`）になっており、ブラウザが新しいファイルをロードする通常の遷移を採用しています。

### 2. セッション管理（`sessionStorage`）によるポップアップ制御
- `index.htm` を最初に開いた際、「デモ案内」のポップアップが表示されます。
- 一度「了解した」ボタンを押すと、JavaScriptがブラウザの `sessionStorage` に `demo_acknowledged: true` というフラグを保存します。
- これにより、他のページ（`kiji.htm` など）へ行ってトップに戻ってきた時や、リロードした時は、ブラウザがフラグを検知してポップアップを非表示（`hidden`）のままにする、賢いUX（ユーザー体験）を実装しています。

### 3. インタラクティブな「農と食スポットマップ」
- 円形のマップ背景と京王線の路線（破線）をCSS/SVGで描画。
- マップ上の各農園ピン（ボタン要素）を `absolute`（絶対配置）でパーセンテージ指定することで、スマホなどの画面サイズが変わっても位置がズレないレスポンシブなマップを実現。
- 各ピンをクリックした際、JavaScriptの `selectSpot` 関数を通じて下部の詳細カードに農園情報や写真を動的に書き換える（`innerHTML`）処理を行っています。

### 4. スタンプログ機能とCSSアニメーション
- スタンプをタップすると、赤い「済」スタンプが表示/非表示されるトグル機能をJavaScriptで実装（`toggleStamp` 関数）。
- スタンプが表示される際、CSSの `@keyframes stamp-pop` アニメーションが発火し、実物の判子を「ポンッ」と押したかのようなスケールアップ＆回転エフェクトを表現しています。

### 5. 外部APIを用いた動的QRコード生成
- カードの左側にあるQRコードは静的な画像ファイルではなく、外部のQRコード生成API（`api.qrserver.com`）に対してURLをパラメータとして渡すことで、**その場で実際のスマートフォンで読み取り可能な本物のQRコード**を動的に生成して表示しています。

## 🛠️ 技術スタック

1. **HTML5**: セマンティックなマークアップによる構造化
2. **JavaScript (ES6+)**: ビューのルーティング、ポップアップ制御、マップ詳細の切り替え
3. **Tailwind CSS v4 (CDN)**: クラス記述による高速かつレスポンシブなスタイリング
4. **Lucide Icons**: わかりやすいベクターアイコンの動的読み込み
5. **Google Fonts (Zen Maru Gothic)**: 親しみやすい丸ゴシックフォントの適用

## 🚀 ローカルでの動作確認方法

特別なWebサーバーや開発ツールのインストールは不要です。
1. 本プロジェクトのフォルダをダウンロードします。
2. `index.htm` をダブルクリックして、お使いのブラウザ（Chrome、Edgeなど）で開くだけで全ての機能が動作します。
